# OpenPlanner v0.1.0

This first release packages OpenPlanner for local agent use through the installed `openplanner` runner and matching Agent Skills-compatible skill.

## Changed

- Added the production `openplanner planning` JSON runner for local calendar and task operations backed by SQLite.
- Added the portable `skills/openplanner` skill payload so agents can use the runner without repository search or direct database access.
- Added local-first calendar and task workflows covering events, tasks, recurrence, reminders, attendees, event-task links, iCalendar import/export, and local backup guidance.
- Added local-only experimental CalDAV compatibility tooling for research while keeping the supported agent deployment path on the JSON runner.
- Added release packaging for platform runner archives, the skill archive, source archive, checksums, SPDX SBOM, installer, and GitHub attestations.

## Verification

- Local repository gates passed with `mise exec -- make check`.
- The release binary smoke verified `openplanner --version` and an unrejected `{"action":"validate"}` planning request.
- The production agent eval gate passed the current full local agent-use scenario suite with recommendation `ship_openplanner_runner_local_agent_use`.
